## Services Used

## Getting started

* Dependency
  - Express JS
  - bcrypt
  - body-parser
  - mongoose
  - passport-facebook
  - passsport-googleAuth
 
  
  
* To install the dependencies.
  - npm install
  
* To run the project.
  -npm start
